Notes de publication¶

Notes de publication pour les publications officielles de Django. Chaque note de publication explique les nouveautés de la nouvelle version ainsi que les modifications non rétro-compatibles effectuées dans cette version.

Pour ceux qui se mettent à niveau à une nouvelle version de Django, ils devront vérifier toutes les modifications non rétro-compatibles ainsi que les fonctionnalités obsolètes de chaque publication « finale » à partir de la version de Django de laquelle ils partent, jusqu’à la nouvelle version sur laquelle ils vont travailler.

Version finale¶

Vous trouverez ci-dessous les notes de publication jusqu’à 3.1 ainsi que les publications correctives. Les nouvelles versions de la documentation contiennent les notes de publication de toute publication antérieure.

Versions de sécurité¶

Lors de chaque notification de problème de sécurité selon la politique de sécurité de Django, des notes de publication correspondantes sont chaque fois ajoutées à toutes les séries de publication affectées.

De plus, une rétrospective de toutes les notifications de sécurité publiées peut être consultée.
