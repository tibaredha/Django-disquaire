Gabarits¶

Le moteur de gabarit de Django offre un mini-langage puissant pour définir la couche visible par les utilisateurs de votre application, encourageant une séparation propre entre la logique applicative et la présentation. Les gabarits peuvent être maintenus par tout connaisseur du langage HTML ; aucune connaissance de Python n’est obligatoire. Pour du contenu d’initiation, consultez le guide thématique Gabarits.
